Netsy by AnthonySteele

A .Net Client library for the API of the website Etsy.com for use in .Net and Silverlight programs

The main aims of Netsy are - Access to the Etsy API from .Net - Works in Silverlight and Regular .Net - Code in C# and .Net 3.5 - Asynchronous service calls. i.e. The service call returns immediately, and a callback is called once data is available. - Client projects in Silverlight to embed rich etsy shop data in web pages. Netsy is far from complete, in fact it’s at the “proof-of-concept with general architecture in place” stage.
