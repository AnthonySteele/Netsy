Netsy is a .Net wrapper for the Etsy website 
Written in C# 3.5
By Anthony Steele
Regular .Net and Silverlight are both supported


Netsy project Home: http://github.com/AnthonySteele/Netsy/

Etsy Website: http://www.etsy.com 
Etsy developer documentation: http://developer.etsy.com/
